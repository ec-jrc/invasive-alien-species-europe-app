Τα χωροκατακτητικά ξένα είδη (IAS) μπορεί να έχουν σοβαρές επιπτώσεις στη βιοποικιλότητα, στα οικοσυστήματα και στις ανθρώπινες δραστηριότητες. Γίνε μέρος της προσπάθειας αντιμετώπισης των χωροκατακτητικών ξένων ειδών! Αυτή η εφαρμογή σας επιτρέπει να αναφέρετε παρατηρήσεις στην περιοχή σας και έτσι να συμβάλλετε στην έγκαιρη ανίχνευση νέων ειδών-εισβολέων.

Η διαδικασία ανάδρασης (feedback) για τις παρατηρήσεις βρίσκεται ακόμη υπό ανάπτυξη και θα είναι διαθέσιμη σε επόμενες εκδόσεις της εφαρμογής.

Ο αναλυτικός οδηγός χρήσης αυτής της εφαρμογής στα Αγγλικά είναι διαθέσιμος στον παρακάτω σύνδεσμο.
